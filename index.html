My Latest Work

Harry Potter Analysis

Python, Jupyter Notebook

Using Jupyter Notebook, I was able to analyze the popular fantasy novels and its film adaptations. I used pandas to extract data from CSV files and PyPDF2 to read the PDF files of the novels. I was able to analyze the differences between the novels and its films which I showcased the significances using plotly to graph the differences. I was also able to dive deeper into the magic spells that were used by each main character in the series.

Bookpedia

React, JavaScript, HTML/CSS

This web application allows the user to search up any book and with each search, the book's description, genres, and photo was provided to the user. Bookpedia was created using React and the Open Library Search API was used to extract information for each book. Further features will be added onto this application, stay tuned!
